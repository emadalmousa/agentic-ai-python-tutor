Die 3 Tools des Agents
📖
explain_code_tool
agent/tools/explain_tool.py

Erklärt den Code auf Deutsch in 3–4 Sätzen. Was macht er, wie funktioniert er, einfache Sprache für Anfänger.

Erklärung
🐛
debug_code_tool
agent/tools/debug_tool.py

Analysiert den Code auf Syntax- und Logikfehler. Gibt Fehlertyp und konkreten Verbesserungsvorschlag zurück.

Fehler finden
🎯
exercise_tool
agent/tools/exercise_tool.py

Generiert eine passende Übungsaufgabe basierend auf dem Code des Studenten — maximal 3 Sätze, direkt anwendbar.

Warum kein Chatbot?

Agent vs. ChatGPT / Chatbot

💬

ChatGPT / Chatbot

Reagiert auf Text — antwortet mit Text

🧠 Verarbeitung

LLM generiert Antwort direkt — kein Werkzeug, keine Zwischenschritte

🔁 Loop

Keiner — eine Frage, eine Antwort, fertig

📤 Ausgabe

Ein freier Textblock — keine feste Struktur, keine spezialisierten Aktionen

VS
🤖

Unser LangChain Agent

Plant, handelt, überprüft — eigenständig

🧠 Verarbeitung

LLM entscheidet welche Tools gebraucht werden — explain · debug · exercise

🔁 Loop

Läuft bis tool_calls leer ist — mehrere Runden möglich

📤 Ausgabe

5 strukturierte Felder: Erklärung · Fehler gefunden · Fehlertyp · Hinweis · Übung

Sprint 2 — Feature
RAG + PDF Upload

Student lädt eigenes Kursmaterial als PDF hoch. Der Agent durchsucht es semantisch — bei jeder Analyse und im Chat wird das Material als Kontext genutzt.

→ PDF → Chunks → FAISS Vector Store
→ Seitenreferenz „Seite 9" direkt abrufbar
→ Semantische + direkte Suche kombiniert
📈
Sprint 2 — Feature
Lernfortschritt-Tracking

Der Tutor merkt sich welche Konzepte der Student bereits geübt hat. Übungen werden gezielt auf Schwächen zugeschnitten — kein Wiederholen was schon sitzt.

→ Session-Historie: gelöste Aufgaben
→ Schwierigkeitsgrad steigt automatisch
→ Fortschrittsanzeige im Frontend
👤
Sprint 2 — Feature
Schülerprofil

Jeder Student bekommt ein persistentes Profil — Lernstand, Schwächen, bevorzugte Erklärungsstile. Der Agent passt Ton und Tiefe automatisch an den Schüler an.

→ Login / Profil-Persistenz
→ Lernstil-Erkennung aus Verlauf
→ Personalisierte Übungen & Erklärungen
